# 서울 아파트 가격 예측 프로젝트

> 2026-1 데이터마이닝 | 한성대학교 컴퓨터공학부 | B분반 4팀 **찹쌀떡**

공공데이터 기반 외부 환경 변수를 직접 설계하고, 변수 그룹별 기여도를 단계적으로 분석하여 서울 아파트 가격 예측 모델을 구축한 프로젝트입니다.

## 실행 방법

### 환경 설정

```bash
pip install pandas numpy scikit-learn matplotlib requests python-dotenv
```

`.env` 파일에 카카오 API 키 추가:

```
KAKAO_API_KEY=your_api_key_here
```

### 데이터 준비

`data/raw/` 폴더에 아래 파일을 위치시킵니다:

```
data/raw/
├── 서울_지하철.csv
├── 서울_버스정류장.csv
├── 서울_공원.csv
├── 서울_학원.csv
├── 서울_의료시설.csv
├── 서울_유흥주점.csv       # 공공데이터포털 다운로드 후 전처리본
└── 서울_모텔.csv           # 공공데이터포털 다운로드 후 전처리본
```

### 노트북 실행 순서

```
1. 01_preprocessing.ipynb   → data/processed/apt_preprocessed.csv 생성
2. 02_features.ipynb        → data/processed/apt_master.csv 생성
3. 03_experiments.ipynb     → 실험 결과 CSV 생성
```

## 모델 설정

| 모델 | 하이퍼파라미터 |
|---|---|
| Ridge 회귀 | alpha=1.0 / StandardScaler 적용 |
| Random Forest | n_estimators=100 / random_state=42 |
| 공통 | Train 80% / Test 20% (random_state=42) |
