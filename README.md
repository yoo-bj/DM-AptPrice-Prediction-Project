# Workspace

각 팀원의 개인 작업 공간. **이 폴더 내용은 깃에 안 올라간다** (`.gitignore` 처리).

## 사용 방법

```bash
# 템플릿 복사해서 자기 노트북 만들기
cp _template.ipynb 김OO_transport.ipynb
```

자기 노트북에서 자유롭게 작업:
- 외부 데이터 탐색
- 변수 생성 테스트
- EDA / 시각화 / AI 활용 분석
- 채택/탈락 판단

## 작업이 끝나면

1. 채택할 변수 생성 코드를 `src/features.py`의 자기 담당 섹션에 함수로 옮김
2. 자기 브랜치에서 PR 생성 → main 머지
3. 본 노트북은 그대로 자기 컴퓨터에 보관 (나중에 정리 후 깃에 올릴 예정)

## 명명 규칙

- 파일명: `{이름}_{담당그룹}.ipynb`
- 예: `김OO_transport.ipynb`, `이OO_education.ipynb`

## 분석 결과 메모

채택/탈락 근거는 **자기 작업 노트**(워드/노션/메모장 등)에 자유 양식으로 기록.
보고서 작성 시 그 노트를 가져와서 보고서에 옮긴다. 별도 md 파일 만들지 않음.
